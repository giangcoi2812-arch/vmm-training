# 🏔️ Kiến Thức Trail Running

> Thư mục chứa kiến thức chuyên sâu về chạy trail — phân biệt với chạy đường nhựa (road running).

## Nội dung sẽ bao gồm:
- Kỹ thuật chạy trail (lên dốc, xuống dốc, địa hình kỹ thuật)
- Hiểu về địa hình & elevation gain
- Trang bị chuyên trail (giày trail, gậy, balo nước...)
- Khác biệt trail vs road running
- An toàn khi chạy trail (thời tiết, động vật, lạc đường...)
